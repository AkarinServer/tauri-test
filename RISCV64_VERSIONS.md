# Node.js RISCV64 可用版本列表

## 重要发现

**Node.js 22 版本中没有 RISCV64 构建！**

根据对 [unofficial-builds.nodejs.org](https://unofficial-builds.nodejs.org/download/release/) 的检查：

- ❌ **Node.js 22.x**: 没有任何版本有 RISCV64 构建
- ✅ **Node.js 21.x**: 有 RISCV64 构建（最新: v21.7.2）
- ✅ **Node.js 20.x**: 有 RISCV64 构建（最新: v20.19.5）

## 下载 URL 格式

正确的下载 URL 格式：
```
https://unofficial-builds.nodejs.org/download/release/v{VERSION}/node-v{VERSION}-linux-riscv64.tar.xz
```

示例：
- Node.js 21.7.2: `https://unofficial-builds.nodejs.org/download/release/v21.7.2/node-v21.7.2-linux-riscv64.tar.xz`
- Node.js 20.19.5: `https://unofficial-builds.nodejs.org/download/release/v20.19.5/node-v20.19.5-linux-riscv64.tar.xz`

## 建议

由于 Node.js 22 没有 RISCV64 构建，建议：

1. **使用 Node.js 21.7.2**（Node.js 21 系列最新版本）
2. **或使用 Node.js 20.19.5**（Node.js 20 LTS 系列最新版本）

这两个版本都有 RISCV64 构建，并且 URL 格式已验证正确。
